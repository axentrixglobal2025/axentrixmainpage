Product Details

Public Print

Public Print revolutionizes document printing by allowing users to easily print documents in public locations such as libraries, cafes, and business centers without the need for personal printers or complex setups.

Key Features:
  • QR Code Printing: Simply scan a QR code at the printing station, upload your document, and print instantly.
  • WhatsApp Integration: Send your document via WhatsApp to a designated number and receive a printing code to use at any Public Print location.
  • Direct WiFi Connection: Connect directly to the printer's secure WiFi network to print without internet access.
  • Multiple File Formats: Support for PDF, DOC, DOCX, JPG, PNG, and other common file formats.
Availability

Currently available in select locations with plans for nationwide expansion in the coming months.

Rin App / Beyazwale

Rin App (Beyazwale) is a digital platform that connects verified local money lenders with customers looking for quick, trusted, and secure lending services, filling the gap between traditional banking and informal lending.

Key Features:
  • Verified Lenders: All lenders on the platform undergo strict verification processes to ensure trust and security.
  • Secure Transactions: End-to-end encrypted communication and secure payment processing for all transactions.
  • Quick Approval Process: Streamlined application and approval process for faster access to funds when needed.
  • Transparent Terms: Clear display of interest rates, fees, and terms for all lending offers.
Availability

Currently available in select cities with ongoing expansion to more regions. Download the app from the App Store or Google Play.
